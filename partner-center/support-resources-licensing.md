# <a name="guide-to-licensing-resources-for-microsoft-partners"></a>Guía para los recursos de licencias de los asociados de Microsoft 

**Roles adecuados:** Agente de ayuda

En este artículo se comparten recursos que pueden ayudar a responder a preguntas sobre licencias de asociados.

Como asociado de Microsoft, puede estar implicado en uno o varios programas de licencias, canales o contratos. Si trabaja en el panel Centro de partners, puede incluir programas como el programa Proveedor de soluciones en la nube (CSP) o el programa Microsoft Partner Network (MPN). Otros programas de licencias incluyen Contrato Enterprise licencias (EA), Licencias por volumen (VL), el programa de licencias de OEM, el programa de entrenamiento o vales de Software Assurance o el programa licencias abiertas.

Cada programa tiene contratos y términos específicos asociados. De forma similar, los productos específicos que use pueden tener contratos de licencia adicionales específicos del producto asociados a su uso.

Si tiene preguntas sobre cómo funcionan estos programas, estos recursos pueden ayudarle.

> [!NOTE]
> Si necesita ayuda con preguntas de licencia más detalladas o con una cita de cliente, comience con el proveedor indirecto asignado del programa de licencias (para el programa CSP) o el distribuidor del programa. A menudo son sus mejores puntos de contacto iniciales para preguntas de licencia específicas o para preguntas sobre la creación de citas de clientes, compras de SKU o requisitos previos. Si no tiene un asociado que atendiendo este rol, consulte Búsqueda de [un asociado](find-a-partner.md) para buscar uno.
>
> También puede comunicarse con los contactos de Microsoft para aprovechar un equipo interno de expertos en licencias. Si no tiene contactos específicos de [](https://partner.microsoft.com/support/v2/?stage=1) Microsoft, vaya a Soporte técnico para partners y escriba **licencias** en el campo Problemas de la **página** Soporte técnico.
