# <a name="core-requirements-for-the-activities-in-your-incentive-program"></a>Requisitos básicos para las actividades de su programa de incentivos

**Roles adecuados**

- Administrador de incentivos
- Usuario de incentivos

Cada tipo de actividad para el que envía una reclamación de incentivos tiene requisitos básicos. Debe demostrar que ha cumplido estos requisitos en los documentos de prueba de ejecución. Algunas actividades también requieren tipos y métricas de comunicaciones específicos.

En las tablas siguientes se describen los requisitos básicos para cada una de las actividades aceptadas.
