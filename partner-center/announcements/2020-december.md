## <a name="an-update-of-the-us-microsoft-365-business-voice-with-calling-plan-offer-is-coming-soon"></a><a name="2"></a>Próximamente tendrá disponible una actualización de la oferta de Microsoft 365 Business Voice con el plan de llamadas para EE. UU.

### <a name="categories"></a>Categories

- Fecha: 03/12/2020
- Ofertas/mercados

### <a name="summary"></a>Resumen 

El 1 de enero de 2021, Microsoft comenzará a migrar a los asociados y clientes de EE. UU. a una nueva oferta de Microsoft 365 Business Voice con el plan de llamadas. No se requiere ninguna intervención de los asociados.

### <a name="impacted-audience"></a>Audiencia afectada 

Todos los partners que operan a través del programa Proveedor de soluciones en la nube (CSP)

### <a name="details"></a>Detalles

Microsoft reemplazará la nueva oferta existente de Business Voice con el plan de llamadas en los EE. UU. por una nueva a fin de cumplir los requisitos internos de los productos de telecomunicaciones. La nueva oferta tendrá el mismo conjunto de características y el mismo precio. Los cambios en la oferta de Business Voice con el plan de llamadas son internos de Microsoft y no deben afectar a la forma en que los asociados de CSP la comercializan, la venden u ofrecen soporte técnico de esta. El cambio solo se aplicará a esta oferta.

En el caso de las nuevas ventas, se usará la nueva oferta a partir del 1 de enero de 2021. La oferta anterior se reemplazará por la nueva en la lista de precios.

Los clientes existentes se migrarán automáticamente a la nueva oferta en función de su fecha de renovación. No es necesario que los asociados y los clientes que realicen la renovación hagan ninguna acción.

## <a name="now-live-partner-center-api-updates-and-user-interface-enhancements-for-the-education-customer-validation-process"></a><a name="1"></a>Ya disponibles: actualizaciones de la API del Centro de partners y mejoras de la interfaz de usuario para el proceso de validación de clientes de Education

### <a name="categories"></a>Categories

- Fecha: 03/12/2020
- Funcionalidades

### <a name="impacted-audience"></a>Audiencia afectada 

Asociados que venden ofertas académicas mediante el programa Proveedor de soluciones en la nube

### <a name="summary"></a>Resumen 

Microsoft funciona con confianza. Nos comprometemos a proporcionar un método compatible, seguro y protegido para la validación de clientes durante las operaciones de transacción con ofertas académicas en el programa Proveedor de soluciones en la nube. Como parte de este compromiso, presentamos la API del Centro de partners y mejoras en la interfaz de usuario que agregarán claridad y visibilidad en el proceso de verificación de los clientes, así como la capacidad de introducir datos más precisos, lo que mejorará la eficacia de la verificación de clientes. 

**Mejoras en el Centro de partners** 

- Nuevas API de calificaciones GET y POST para admitir la entrada de datos precisa y mejorar el proceso de validación de clientes de Educación de Microsoft. 

- Mejoras en la interfaz de usuario para admitir la entrada de datos precisa y mejorar el proceso de validación de clientes de Educación de Microsoft. 

**Desuso de las API de calificación GET y PUT existentes** 

Las API de calificación GET y PUT existentes se retirarán **antes del final de febrero de 2021**. En ese momento, deberá haber realizado la transición a las nuevas API GET y POST del Centro de partners a fin de poder comprar ofertas de Education.  

**Pruebas**: 

Para comprender mejor las API y la entrada de datos necesarias para una correcta validación de los clientes, los asociados podrán probar estas mejoras. Los asociados que quieran participar en las pruebas deberán descargar la [guía de pruebas para clientes de Education del Centro de partners](https://partner.microsoft.com/resources/detail/partner-center-edu-testing-guide-pdf) para obtener información sobre cómo prepararse, cómo registrarse y qué esperar durante la fase de pruebas.

**Clientes de bibliotecas y museos** 

Además de las mejoras anteriores, también nos complace anunciar que habilitaremos ofertas con precios académicos para clientes de bibliotecas y museos, lo que permitirá ampliar la base de clientes de Educación con los que puede realizar transacciones de ofertas de CSP. 

Microsoft se reserva el derecho de revisar el estado de cualquier cliente real o propuesto como usuario educativo cualificado. Para obtener toda la información, consulte los [requisitos para usuarios académicos cualificados de Education](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7). 

### <a name="next-stepsadditional-resources"></a>Pasos siguientes/recursos adicionales

- Revise la nueva interfaz de usuario del Centro de partners, los cambios en la API y la guía en los recursos de preparación de operaciones:  Mejoras en el proceso de [validación de clientes de Education del Centro de partners](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements#/) 

- Asegúrese de que su organización esté familiarizada con los [requisitos para usuarios académicos cualificados de Educación](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7). 

- Comparta esta información con los equipos adecuados dentro de su organización y con sus revendedores para ayudarlos a prepararse para estos cambios.
