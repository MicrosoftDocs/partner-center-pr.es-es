## <a name="partner-center-api-updates-and-user-interface-enhancements-for-the-education-customer-validation-process"></a><a name="8"></a>Actualizaciones de la API del Centro de partners y mejoras de la interfaz de usuario para el proceso de validación de clientes de Educación

### <a name="categories"></a>Categories

- Fecha: 31-08-2020
- Funcionalidades

### <a name="impacted-audience"></a>Audiencia afectada

Partners que venden ofertas académicas a través del programa Proveedor de soluciones en la nube (CSP) 

### <a name="summary"></a>Resumen

Microsoft funciona gracias a la confianza y se compromete a proporcionar un método compatible, seguro y protegido para la validación de clientes durante las operaciones con ofertas académicas en el programa Proveedor de soluciones en la nube. Como parte de este compromiso, presentamos la API del Centro de partners y mejoras en la interfaz de usuario en el segundo trimestre de este año fiscal (2021) que agregarán claridad y visibilidad en el proceso de verificación de los clientes, así como la capacidad de introducir datos más precisos, lo que mejorará la eficacia de la verificación de clientes. 

#### <a name="partner-center-enhancements"></a>Mejoras en el Centro de partners

- Nuevas API de calificaciones GET y POST para admitir la entrada de datos precisa y mejorar el proceso de validación de clientes de Educación de Microsoft.
- Mejoras en la interfaz de usuario para admitir la entrada de datos precisa y mejorar el proceso de validación de clientes de Educación de Microsoft.

#### <a name="testing"></a>Prueba

Los partners tendrán la oportunidad de probar estas mejoras a partir de octubre. Recomendamos encarecidamente aprovechar esta oportunidad para obtener una mejor comprensión de las API y de la entrada de datos necesarias para una correcta verificación de los clientes. Pronto se proporcionarán más detalles sobre los plazos exactos y cómo participar. Las API existentes del Centro de partners se retirarán antes del final del segundo trimestre del año fiscal 2021, momento en el que todos los partners deberán pasar a usar las nuevas API del Centro de partners.

#### <a name="library-and-museum-customers"></a>Clientes de bibliotecas y museos

Además de las mejoras anteriores, también estamos encantados de anunciar que en el segundo trimestre del año fiscal 2021 habilitaremos ofertas con precios académicos para clientes de bibliotecas y museos, lo que permite ampliar la base de clientes de Educación a los que puede extender ofertas de CSP.

Microsoft se reserva el derecho de revisar el estado de cualquier cliente real o propuesto como usuario educativo cualificado. Para obtener toda la información, consulte los [requisitos para usuarios académicos cualificados de Educación](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7).

### <a name="next-stepsadditional-resources"></a>Pasos siguientes y recursos adicionales

- Revise la nueva interfaz de usuario del Centro de partners y los cambios en la API en los recursos de Preparación de operaciones: [Mejoras en el proceso de validación de clientes de Educación del Centro de partners](https://partner.microsoft.com/resources/collection/partner-center-edu-validation-enhancements#/)
- Asegúrese de que su organización esté familiarizada con los [requisitos para usuarios académicos cualificados de Educación](https://www.microsoftvolumelicensing.com/DocumentSearch.aspx?Mode=3&DocumentTypeId=7).
- Comparta esta información con los equipos adecuados dentro de su organización y con sus revendedores para ayudarlos a prepararse para estos cambios.
- Habrá más novedades en la oportunidad de pruebas de octubre, donde se proporcionarán detalles sobre los plazos exactos y cómo participar.

## <a name="sdk-release-on-net-standard-v1161"></a><a name="6"></a>Versión del SDK en .NET Standard (v1.16.1)

### <a name="categories"></a>Categorías

- Fecha: 12-08-2020
- Funcionalidades

### <a name="impacted-audience"></a>Audiencia afectada

Partners de factura directa y proveedores indirectos que participan en el programa CSP y que usan el SDK de .NET del Centro de partners. 

### <a name="summary"></a>Resumen

A partir del 21 de julio de 2020, los partners pueden empezar a descargar la versión del **SDK de .NET Standard del Centro de partners de Microsoft**, SDK de .NET v1.16.1, junto con los **ejemplos actualizados públicos del SDK del Centro de partners de GitHub**.

### <a name="details"></a>Detalles

Los partners de facturación directa y los proveedores indirectos que participan en el programa CSP pueden descargar la versión más reciente del SDK de .NET Standard. Estos cambios harán que el SDK sea compatible con la aplicación existente mediante el .NET Framework 4.6.1 y versiones posteriores. La plataforma .NET Standard será compatible con .NET Core 2.0 y versiones posteriores.

Una gran cantidad de partners ha solicitado el SDK de .NET Standard. La versión actual del SDK obliga a los partners a usar aplicaciones escritas en ASP.NET Framework, que es una solución lenta y costosa en comparación con las aplicaciones de .NET Core. Por lo tanto, la migración a .NET Standard puede ayudar a resolver los problemas de rendimiento o compatibilidad con .NET Core.
 
**Aspectos destacados**
- No hay más versiones de Microsoft para .NET Framework 4.8, ya que es la última de la línea.
- Limita el desarrollo futuro, ya que la mayoría de las bibliotecas se publicarán para .NET Core o .NET Standard, que podrían no ser compatibles con la versión del marco actual.
- .NET Core es conveniente para su uso con contenedores debido a su naturaleza ligera.
- El entorno de tiempo de ejecución de alto rendimiento del servidor para Windows Server y Linux convierte a .NET Core en un marco web de rendimiento superior.

### <a name="next-steps"></a>Pasos a seguir

- Descargar la última versión del [SDK de .NET v1.16.1](https://www.nuget.org/packages/Microsoft.Store.PartnerCenter/1.16.1)
- Versión: SDK de .NET v1.16.1
- Código de ejemplo: [Código de ejemplo](https://github.com/microsoft/Partner-Center-DotNet-Samples)
- GA: 12/08/2020
- No existen nuevos cambios en la API en la versión actual

## <a name="announcing-the-availability-of-visio-plan-1-and-plan-2-trial-offers"></a><a name="1"></a>Anuncio de la disponibilidad de las ofertas de prueba de Visio Plan 1 y Plan 2 

### <a name="categories"></a>Categories

- Fecha: 03-08-2020
- Ofertas/mercados

### <a name="impacted-audience"></a>Audiencia afectada

Todos los partners que participan en el programa Proveedor de soluciones en la nube (CSP)

### <a name="summary"></a>Resumen

Microsoft presenta ofertas de prueba para Visio Plan 1 y Plan 2 a partir del 1 de agosto de 2020. 

### <a name="details"></a>Detalles

Microsoft Visio permite a los usuarios crear diagramas profesionales sin esfuerzo, colaborar de forma más eficaz, conectarse a datos en tiempo real y automatizar los flujos de trabajo. Las ofertas de prueba de Visio Plan 1 y Plan 2 estarán disponibles a partir del 1 de agosto de 2020.

Microsoft Visio puede usarse en una gran variedad de sectores y roles con integración en soluciones de Power Platform, Microsoft 365 y Microsoft Teams. Ofrezca a los clientes las herramientas que necesitan para llevar a cabo sus negocios.

#### <a name="what-are-the-partner-benefits"></a>¿Cuáles son las ventajas de los partners?

Microsoft Visio es un producto que le ayudará a:

- Usar escenarios de integración y plantillas de gráficos organizativos con Excel y Power BI para compilar una estructura organizativa sostenible.
- Planear una vuelta segura al área de trabajo con planos de Visio para incorporar instrucciones de distanciamiento social.
- Adaptar los modelos empresariales: el análisis de negocio es uno de los aspectos fundamentales de la propuesta de valor de Visio. Adapte los modelos empresariales a la nueva forma de trabajar mediante la asignación y el rediseño de los procesos de negocio críticos con Visio y obtenga información oportuna sobre los datos con los diagramas de Visio y Power BI integrados en esa aplicación. 
- Centrarse en la creación de valor y la reducción de costes: identifique y automatice los procesos repetitivos y ayude a los clientes a desarrollar cargas de trabajo de autoservicio para roles funcionales comunes, como administración de TI, recursos humanos y otros.

Ofrecemos una prueba gratuita de 30 días para Visio Plan 1 y Plan 2 para ayudar a los clientes a experimentar estas ventajas. Los partners pueden encontrar las pruebas en la lista de precios mensual que aparece en la página de precios y ofertas del Centro de partners. Las ofertas de prueba indicarán "PRUEBA" en la columna Tipo de licencia secundaria de la lista de precios.
